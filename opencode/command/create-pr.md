# Create PR

1. Read the current branch name and recent commits
2. Generate a descriptive PR title and body from the commits. Use the .github/PULL_REQUEST_TEMPLATE.md or other pull request templates, if they exist.
3. Present a draft of the PR body to me for approval.
4. Upon receiving approval, use `gh pr create` with the generated title/body
5. Assign to me and add relevant reviewers if specified
